## 🔧 Embedded LLM Prompt Template (for live deployment)

The prototype renders briefs deterministically. To go live, replace the `render_executive_brief` function body with a single LLM call using this template:

```
You are a senior CRM strategist. Write a 6-bullet executive brief for the 'VIP B2B Accounts' segment using ONLY the data provided. Tone: CMO-ready, no jargon, every claim tied to a number. Structure: who they are, why they matter, top risk, top opportunity, recommended campaign, named KPI to measure success.
```

Inputs available to the prompt: `summary`, `alert`, `opportunity`, `recommendation` (all structured dicts).
