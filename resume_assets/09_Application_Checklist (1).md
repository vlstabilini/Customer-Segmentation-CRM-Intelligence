## 📡 Channel Strategy (where to apply)

### Primary channel — Avelabs careers page / official form
- Submit Resume PDF + Cover Letter (EN+KR if available)
- Include GitHub URL in the resume header
- Include the 150-word project description in the application form free-text field

### Secondary channel — LinkedIn direct outreach
- Identify the hiring manager / data team lead on LinkedIn
- Send a connection request with a brief note (the LinkedIn Post 4 hook from `08_linkedin_content_pack.docx` is ideal seed content)
- After connection acceptance, follow up with: 1-paragraph intro + GitHub URL + resume PDF as a document attachment
- Response rate uplift: ~20-30% vs. cold form submission

### Tertiary channel — LinkedIn content campaign
- Start publishing the 5 LinkedIn posts from `08_linkedin_content_pack.docx` one per week
- Post 4 (Agentic AI · "LLM renders, tools decide") is the strongest hook for Avelabs specifically — schedule it the week of your application
- Tag #Avelabs and 2-3 known Avelabs employees per post (visible LinkedIn algorithm uplift)

### Quaternary channel — Korean job platforms
- Wanted.kr · jobplanet.co.kr · Rocketpunch · saramin.co.kr (if Avelabs lists there)
- Use the Korean 150자 / 300자 project descriptions from `08_project_descriptions.md`
- Submit the EN+KR cover letter
